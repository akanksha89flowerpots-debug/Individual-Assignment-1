# Individual-Assignment-1

# PROJECT: BMI Calculator & Vehicle Route Simulation



# OVERVIEW

# This project contains two independent console-based programs:
#
# 1) BMI (Body Mass Index) Calculator
# 2) Vehicle Route Distance Simulation
#
# Both programs focus on basic programming concepts such as:
# - User input handling
# - Mathematical calculations
# - Unit conversions
# - Logical comparisons and decision-making




# PROBLEM 1: BMI CALCULATOR


# PURPOSE:
# The BMI Calculator determines whether a person’s weight is
# within a healthy range relative to their height.

# FUNCTIONALITY:
# - Prompts user for weight in pounds
# - Prompts user for height in inches
# - Calculates BMI using the imperial formula:
#       BMI = (weight / height^2) * 703
# - Converts inputs into metric units:
#       weight (kg) = pounds × 0.453592
#       height (m)  = inches × 0.0254
# - Calculates BMI using metric formula:
#       BMI = weight / height^2
# - Displays both BMI results rounded to 2 decimal places

# KEY CONCEPTS:
# - Unit conversion (imperial ↔ metric)
# - Arithmetic operations and exponentiation
# - Floating-point formatting


# PROBLEM 2: VEHICLE ROUTE SIMULATION


# PURPOSE:
# This program simulates travel distance based on speed and time
# under ideal driving conditions (no traffic or delays).

# FUNCTIONALITY:
# - Uses or accepts travel time (hours)
# - Uses or accepts average speed (miles per hour)
# - Computes distance using:
#       Distance = Speed × Time
# - Evaluates multiple route options:
#       Local, Parkway, Highway
# - Calculates distance for each route
# - Compares results to determine shortest distance

# ROUTE DATA:
# - Local   → 1.00 hours at 30 mph
# - Parkway → 0.88 hours at 40 mph
# - Highway → 0.87 hours at 55 mph

# KEY CONCEPTS:
# - Distance calculation
# - Data comparison
# - Minimum value selection
# - Basic simulation modeling


# LEARNING OUTCOMES

# This project demonstrates:
# - Basic programming structure
# - Input/output operations
# - Mathematical modeling of real-world problems
# - Unit conversions
# - Decision-making using comparisons
